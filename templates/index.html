3Ligand filtering & 3D prep — applied to each generated batch before docking

Pipeline per step: ligfilter drops unwanted molecules (they get reward 0, so the agent learns to avoid them) → an RDKit ETKDG 3D conformer is embedded → GNINA docks them. Optionally enable ligprep for chemically richer 3D states (salt-strip, tautomer/protomer at pH via cxcalc, stereo, CONFORGE) instead of the single ETKDG conformer.

Blank = filter off. Format MIN:MAX, MIN:, or :MAX.

Default (off): a single RDKit ETKDG conformer per molecule. When on: multiple states are docked and the best per molecule is kept — RL-safe caps apply (stereo enumeration capped at 4, CONFORGE timeout 20 s) so a single molecule can't stall a step.

REINVENT's prior often emits minor tautomers (e.g. exocyclic-imine pyridines N=c1cccc[nH]1, lactim heterocycles). When on, each molecule is normalized to its dominant tautomer before docking, and the results grid / CSV export show the canonical form (raw SMILES kept in the CSV too). Turn off to dock exactly what was generated.
